# Install Required Dependencies

## Frontend Dependencies

Run these commands in the `/app` directory:

```bash
cd app

# Install OpenAI SDK for call analysis
npm install openai

# Install date-fns for date formatting
npm install date-fns

# Verify installation
npm list openai date-fns
```

## Verify Installation

After installation, the following should work:
- ✅ AI call analysis with OpenAI GPT-4
- ✅ Date formatting in call history
- ✅ Post-call summary generation
- ✅ Analytics dashboard
