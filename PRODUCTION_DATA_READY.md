## 🎨 Template Variables in Scripts

All scripts support these variables:

- `{{firstName}}` - Lead's first name
- `{{lastName}}` - Lead's last name
- `{{company}}` - Lead's company
- `{{jobTitle}}` - Lead's job title
- `{{repName}}` - SDR name (defaults to "Alex")

**Example:**

```
"Hey {{firstName}}, this is {{repName}} from Atomicwork..."
→ "Hey Rajesh, this is Alex from Atomicwork..."
```
